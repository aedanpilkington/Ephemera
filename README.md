# Ephemera
CSC3231 Coursework
When in unity, You should be on Ephemera Main. Go to File -> Build Settings -> Add Open Scenes -> Then add in this order Ephemerea Main, TreeInterior_2D, EphemeraButterfly. -> Then Click build and choose the EphemeraBuild Folder. Then the .exe should build in that folder so locate that folder on your PC now and run the .exe file
